# Отчёт: веб-API MiniCRM Booking (JavaScript)

## Цель
Реализовать веб-сервис (API), принимающий запросы и выполняющий бизнес-логику, согласованную с практическими работами 1 и 2 (сервис записи клиентов, подбор свободных слотов, хранение записей).

## Технологии
- Node.js (JavaScript, ES-модули)
- Express — маршрутизация HTTP, JSON

## Реализованные методы

### GET (получение данных)
| Метод | Назначение |
|--------|------------|
| `GET /api/health` | Проверка работоспособности API |
| `GET /api/services` | Список услуг (демо-данные) |
| `GET /api/appointments` | Список созданных записей (в памяти) |

### POST (отправка данных и расчёт)
| Метод | Назначение |
|--------|------------|
| `POST /api/slots/suggest` | Подбор свободных слотов по окну поиска, длительности, буферу и переданным занятым интервалам (`busyIntervals`) — аналог описанного в ПР1/ПР2 эндпоинта |
| `POST /api/appointments` | Создание записи с данными клиента (имя, телефон, время начала/конца) |

## Запуск
1. Установить зависимости: `npm install`
2. Запуск: `npm start`
3. По умолчанию порт: `3000`

## Примеры запросов

**Подбор слотов (POST):**
```http
POST http://localhost:3000/api/slots/suggest
Content-Type: application/json

{
  "durationMinutes": 60,
  "windowStart": "2026-04-05T09:00:00.000Z",
  "windowEnd": "2026-04-05T18:00:00.000Z",
  "bufferMinutes": 10,
  "busyIntervals": [
    { "start": "2026-04-05T12:00:00.000Z", "end": "2026-04-05T13:00:00.000Z" }
  ]
}
```

**Создание записи (POST):**
```http
POST http://localhost:3000/api/appointments
Content-Type: application/json

{
  "clientName": "Иванов И.И.",
  "clientPhone": "+79001234567",
  "serviceId": "svc-1",
  "start": "2026-04-05T10:00:00.000Z",
  "end": "2026-04-05T11:00:00.000Z"
}
```

**Получение услуг (GET):**
```http
GET http://localhost:3000/api/services
```

## Примечание
Интеграция с Google Calendar в этом учебном API не вызывается по сети: занятость передаётся в теле `POST /api/slots/suggest` как `busyIntervals` (как если бы её вернул адаптер календаря). Для продакшена подключается отдельный модуль к Google Calendar API.
